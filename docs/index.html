Sprint Engine 2.0 Toolset

About VanToolstic

VanToolstic is a set of tools designed for working with games running on the "Sprint 2.0" engine, created by Presto Studios.
While the tools on this page will likely work with all 3 of the games that utilise the Sprint Engine, the primary focus is the 2002
Xbox exclusive "Whacked!"



Why?

Development of VanToolstic was started by me, Jill, after talking with the composer of Whacked!, Jamey Scott. During
the conversation, Jamey mentioned the very first iteration of Whacked! known as "Boing". This left me wondering
if there was any leftover content from "Boing" still in the final game.

In my pursuit of hidden goodies, I quickly solved the Sprint Engine's Sprint PAK format and utilising Kaitai Struct
was able to easily write a Python extractor for the format. This led to the creation of VanToolstic's first tool, the PAKUnWhacker.

The end goal of VanToolstic is to have a set of software fully capable of extracting\viewing the content from Whacked!. Unfortunately,
it is extremely unlikely that any tools can or will be developed to reinsert this content into the game due to technical limitations and
lack of any real motivation to do so.



List of tools:

PAKUnWhacker

SCBtoPNG

PAKUnWhacker

The PAKUnWhacker is used to extract files from Sprint Engine PAK files. This is the Sprint Engine's primary archive format.
In Whacked! PAK files are used to store some movie, texture, bitmap and sound files amongst other things.

Download: GitHub release page.

SCBtoPNG

Created by JuanJPro

The SCBtoPNG tool is used to convert Sprint Colour Bitmap (SCB) files to PNG files. This is the format used in Sprint Engine games for GUI art.

Download: GitHub release page.
